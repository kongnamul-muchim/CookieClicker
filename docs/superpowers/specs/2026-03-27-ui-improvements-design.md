# 쿠키 클리커 UI 개선 디자인 문서

> **작성일:** 2026-03-27

## 개요

쿠키 클리커 게임의 UI/UX를 개선하여 사용자 경험 향상

## 선택된 기능

### 1. 쿠키 클릭 애니메이션
- 클릭 위치에서 파티클 효과
- 쿠키 부스러기 (10개) + 반짝이 (4개)
- 중앙 정렬로 시작, 바깥으로 퍼짐
- 애니메이션 끝 상태 고정 (fill: forwards)

### 2. 업그레이드 카드 개선
- 각 업그레이드에 고유 이모지 아이콘
- 구매 가능 시 녹색 테두리 + 반짝임 효과 (shimmer)
- 호버 시 카드 살짝 떠오름
- 구매 시 레벨업 애니메이션 (확대 + 파티클)

### 3. 다크모드
- 소프트 퍼플 테마 (눈 편안한 보라빛)
- 낮은 대비, 부드러운 색상
- 토글 버튼으로 라이트/다크 전환
- 설정은 localStorage에 저장

### 4. 진행바/달성률
- 각 카드에 구매 진행바 (현재 쿠키 / 구매 비용)
- 레벨 진행바 (현재 레벨 / 만렙)
- 마일스톤 알림 (강화/특별강화 가능 시)
- 만렙/강화 달성 시 축하 애니메이션

### 5. 모바일 반응형
- 화면 크기에 따른 레이아웃 자동 조절
- 모바일에서 버튼 크기 증가
- 터치 친화적 간격
- 폰트 크기 자동 조절

### 6. 사운드 효과
- 쿠키 클릭음 (딸깍)
- 배경음악 (편안한 BGM)
- Web Audio API 사용 (외부 파일 불필요)
- ON/OFF 토글, 볼륨 조절

## 색상 팔레트

### 라이트모드 (현재)
- 배경: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
- 카드: white
- 강조: #764ba2

### 다크모드 (소프트 퍼플)
- 배경: #1a1825
- 카드: #211f2d
- 텍스트: #b8a9c9
- 강조: #c9bed8
- 진행바: #764ba2

## 파일 구조

```
public/
├── index.html      # 토글 버튼 추가, 레이아웃 수정
├── style.css       # 다크모드, 반응형, 애니메이션 스타일
├── game.js         # 파티클, 진행바, 다크모드 로직
└── sounds.js       # 사운드 관리 (새 파일)
```

## 기술 스택

- CSS: CSS 변수, 미디어 쿼리, 애니메이션
- JavaScript: Web Audio API, Intersection Observer, localStorage
- 반응형: Flexbox, CSS Grid, 미디어 쿼리

## 호환성

- 모던 브라우저 (Chrome, Firefox, Safari, Edge)
- 모바일 브라우저 지원
- 터치 디바이스 지원
