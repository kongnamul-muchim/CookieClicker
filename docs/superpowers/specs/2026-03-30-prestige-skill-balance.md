## 2. 효과 중첩 방식

### 곱셈 중첩 (Multiplicative)

**전체 보너스 (All Bonus):**
```
baseEffect × (1 + sum of % bonuses)

예시:
- 별의 가호(T5): +10%
- 1등급 별(T10): +5%
- 2등급 별(T10): +10%
- 지배자의 왕관(T11): +15%
- 화염의 용(T13): +25%
- 태양 에너지(T14): +30%
- 무한 우주(T15): +40%
- 전능(T17): +50%
- 천사의 축복(T18): +60%
- 대마법사(T18): +50%
- 신왕(T19): +80%
- 초월자(T20): +100%

합계: 465%
→ baseEffect × 5.65배
```

### 덧셈 중첩 (Additive)

**확률 효과:**
```
baseChance + sum of %p additions

예시:
- 크리티컬(T3): 기본 5%
- 행운(T4): +10%p = 15%
- 마법의 유니콘(T13): +20%p = 35%
- 하늘의 독수리(T13): +10%p = 45%
- 제우스의 번개(T16): → 100% (최대)
- 전지(T17): 모든 확률 → 100%
```

### 곱셈 후 덧셈 (Multiply-Then-Add)

**클릭 데미지:**
```
finalClick = baseClick × (1 + sum of % bonuses) × multiplierEffects

% 보너스 (중첩):
- 클릭 강화 I(T2): +10%
- 힘의 반지(T11): +20%
- 전설의 검(T12): +40%
- 초신성 폭발(T15): +100%
- 전력(T17): +200%
- 신왕자(T19): +150%
- 전체 보너스(465%): +465%

합계: 985%
→ baseClick × 10.85배 (패시브)

배율 효과 (곱셈):
- 크리티컬(T3): ×3
- 정확한 타격(T7): ×1.5 → ×4.5
- 번개(T5): ×10 (1% 확률)
- 잭팟(T7): ×100 (0.1% 확률)
- 카니발(T6): ×(1~5) 랜덤
- 운명의 주사위(T8): ×(1~6) 랜덤
```

### CPS 계산

```
finalCPS = baseCPS × (1 + sum of % bonuses) + flatAdditions

% 보너스 (중첩):
- 생산력 강화 I(T2): +10%
- CPS 위성(T9): +25%
- 은하력(T9): 최대 +50% (10건물)
- 지혜의 목걸이(T11): +20%
- 왕의 사자(T13): +50%
- 블랙홀(T15): +100%
- 창조의 성운(T15): +80%
- 신속(T17): +200%
- 신여왕(T19): +150%
- 전체 보너스(465%): +465%

합계: 1170%
→ baseCPS × 12.7배

고정 추가:
- 대지의 기운(T14): +100 CPS
```

## 3. 확률 스킬 시뮬레이션

### 기본 확률 → 행운 → 유니콘 → 전지

```
| 스킬           | 기본 확률 | 행운(+10%p) | 유니콘(+20%p) | 전지(100%) |
|----------------|-----------|-------------|---------------|------------|
| 크리티컬       | 5%        | 15%         | 35%           | 100%       |
| 번개           | 1%        | 11%         | 31%           | 100%       |
| 더블 찬스      | 3%        | 13%         | 33%           | 100%       |
| 잭팟           | 0.1%      | 10.1%       | 30.1%         | 100%       |
| UFO 납치       | 5%        | 15%         | 35%           | 100%       |
| 별의 반짝임    | 10%       | 20%         | 40%           | 100%       |
| 변화의 가면    | 10%       | 20%         | 40%           | 100%       |
```

### 전지 해금 후 기대값

```
클릭 1회 기대값 (전지 해금):
- 크리티컬 ×4.5 (100%)
- 더블 ×2 (100%)
- 번개 ×10 (100%)
- 잭팟 ×100 (100%)

모든 확률이 100%면 → 항상 발동
하지만 다중 확률 스킬은 "각각" 발동

기대값 = baseClick × 4.5 × 2 × 10 × 100 × ...
→ 비현실적 → 제한 필요
```

### 확률 스킬 제한 규칙

```
1. 동일 타입 확률은 "가장 높은 것만" 적용
   - 크리티컬 배율: ×3, ×4.5, ×10, ×100 중 최고 = ×100
   
2. 추가 확률은 "각각" 적용
   - UFO 납치(+50%): 별도 계산 → 추가 50%
   - 별의 반짝임(+50%): 별도 계산 → 추가 50%

3. 전지 해금 시:
   - 잭팟 항상 발동 → ×100
   - UFO 납치 항상 발동 → 추가 ×1.5
   - 별의 반짝임 항상 발동 → 추가 ×1.5
   
   최종: baseClick × 100 × 1.5 × 1.5 = ×225
```

## 5. 이자 시스템

### 계산
```
baseInterest × (1 + sum of %p)

기본:
- 쿠키 이자(T3): +0.1%
- 달의 인력(T14): +0.2%
- 신공주(T19): +1%

합계: 1.3%
→ 보유 쿠키 × 0.013 / 초

예시:
- 1억 쿠키 보유 → 초당 130만 쿠키 이자
```

## 6. 할인 시스템

### 계산
```
finalCost = baseCost × (1 - sum of % discounts)

기본:
- 비용 할인 I(T2): -5%
- 불굴의 방패(T12): -10%

합계: -15%
→ baseCost × 0.85

특별 강화 할인(T4):
- 특별 강화 비용 -20%
→ 특별 강화 비용 × 0.65 (기본 할인 포함)
```

## 7. 자동 클릭 시스템

### 계산
```
baseAutoClick × (1 + sum of % bonuses) × multiplierEffects

기본:
- 자동 클릭(T3): 1회/초
- 천리의 활(T12): +2회/초

합계: 3회/초

각 자동 클릭에:
- 클릭 데미지 % 보너스 적용

- 버닝은 "수동 클릭만" 적용 (연속성 필요)
```

## 8. 최대 효과 시뮬레이션 (모든 스킬 해금)

### CPS
```
baseCPS = 100 (커서만)

전체 해금 후:
- % 보너스: 1170% → ×12.7
- 고정 추가: +100 CPS

finalCPS = (100 + 건물CPS) × 12.7 + 100

포털(5,000,000 CPS) 해금 시:
= (100 + 5,000,000) × 12.7 + 100
= 63,500,127 CPS

이자: 63.5억 쿠키 × 0.013 = 초당 82.5만 쿠키
→ CPS에 추가
```

### 클릭 데미지
```
baseClick = 1 (CPS의 1%)

CPS = 63,500,127 → baseClick = 635,001

전체 해금 후:
- % 보너스: 985% → ×10.85
- 전체 보너스: ×5.65

패시브: 635,001 × 10.85 × 5.65 = 38,913,516

확률 스킬 (전지):
- 잭팟 ×100
- UFO ×1.5
- 별 반짝임 ×1.5

final = 38,913,516 × 100 × 1.5 × 1.5 = 8,760,243,600
→ 87억 쿠키 / 클릭
```

### 버닝 최대
```
+420% × 5.65 = +2373%

클릭 데미지 × 24.73
= 38,913,516 × 24.73 = 962,376,434
→ 9.6억 쿠키 (버닝 최대 + 잭팟)
```

## 9. 밸런스 검토

### 문제점

**1. 전지(T17) + 잭팟(T7) = 항상 ×100**
- 해결: 전지는 "확률 증가"만, 최대 100% 캡
- 잭팟은 0.1% → 전지로 100% 가능 → 너무 강력

**2. CPS × 클릭의 1%**
- CPS 63억 → 클릭 630만 → ×100 = 63억/클릭
- 클릭이 CPS보다 강해짐

**3. 이자 + CPS 시너지**
- CPS 증가 → 이자 증가 → 무한 성장 가능

### 수정 제안

```
1. 전지 제한:
   - 확률 최대 80% (100% 미달)
   - 잭팟만 제외 (항상 100% 금지)

2. 클릭 기반 변경:
   - baseClick = CPS의 0.1% (1% → 0.1%)
   - CPS 63억 → 클릭 630만 → ×100 = 63억

3. 이자 캡:
   - 이자 최대 CPS의 10%
   - CPS 63억 → 이자 최대 6.3억

4. 잭팟 변경:
   - ×100 → ×50 (50배)
```

## 10. 최종 밸런스 (수정 후)

### CPS
```
baseCPS: 건물 합계
% 보너스: 1170%
고정 추가: +100
→ finalCPS = baseCPS × 12.7 + 100

포털만: 5,000,000 × 12.7 = 63,500,000 CPS
```

### 이자
```
이자율: 1.3%
캡: CPS의 10%
→ min(보유쿠키 × 0.013, CPS × 0.1)

CPS 63.5억 → 이자 최대 6.35억/초
```

### 클릭
```
baseClick = CPS × 0.1% = 63,500

% 보너스: 985%
→ 63,500 × 10.85 = 689,175

전체 보너스: ×5.65
→ 689,175 × 5.65 = 3,895,594

확률 스킬 (전지, 80% 캡):
- 잭팟 ×50 (80% 확률)
- 번개 ×10 (80% 확률) → 둘 중 하나만

기대값: 3,895,594 × (0.8 × 50 + 0.2 × 1) = 156,613,522
→ 1.5억/클릭 기대값
```

### 버닝
```
최대 +420% × 5.65 = +2373%
→ ×24.73

버닝 최대 클릭: 3,895,594 × 24.73 = 96,466,729
→ 9,600만/클릭
```

## 11. 프레스티지 ⭐ 밸런스

### 획득량
```
기본: 강화 1회 = 10⭐

리셋 보너스(T6): ×1.5
악마의 계약(T18): ×2

최대: 10 × 1.5 × 2 = 30⭐/강화
```

### 필요 강화 횟수
```
| 목표          | 비용   | 기본(10⭐) | 최대(30⭐) |
|---------------|--------|------------|------------|
| T1~T5         | 350⭐  | 35회       | 12회       |
| T1~T10        | 1455⭐ | 146회      | 49회       |
| T1~T15        | 3875⭐ | 388회      | 129회      |
| 전체(9610⭐)  | 9610⭐ | 961회      | 321회      |
```

### 강화까지 필요 쿠키
```
강화 = 10레벨 달성

업그레이드 레벨 1→10:
- 커서: 15 × 1.15^10 ≈ 61
- 할인(-15%): 61 × 0.85 ≈ 52

첫 강화까지: ~100 쿠키

레벨 10→20 강화:
- 커서 비용: 61 × 1.15^10 ≈ 248
- 할인: ≈ 212

레벨 100 특별 강화:
- 커서 비용: ×5배
- 특별 할인(-20%): ×0.8
- 할인(-15%): ×0.85
- → ×4.25

예시:
- 커서 100레벨 → 특별 강화 비용 ~248 × 4.25 ≈ 1054
```
