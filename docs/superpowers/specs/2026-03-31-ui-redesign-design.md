# UI 개편 디자인 문서

## 변경 날짜
2026-03-31

## 목표
메인 화면을 쿠키 중심으로 단순화하고, 각 기능을 별도 모달로 분리하여 UI를 개편한다.

## 디자인

### 1. 메인 화면
```
┌─────────────────────────────┐
│  쿠키 클리커        🌙 ⚙️   │  ← 헤더 (테마 + 설정)
├─────────────────────────────┤
│  쿠키: 0                   │  ← 상단: 쿠키 수만
│                             │
│                             │
│         🍪 (클릭)           │  ← 쿠키 버튼 (중앙)
│                             │
│                             │
├─────────────────────────────┤
│  클릭당: 1   │   초당: 0   │  ← 구석: 통계
├─────────────────────────────┤
│     🛒 상점      🌳 스킬    │  ← 하단 버튼
└─────────────────────────────┘
```

**구성:**
- 상단: 쿠키 수 (큰 폰트)
- 중앙: 쿠키 버튼
- 하단 구석: 클릭당/초당 통계
- 하단 버튼: 상점 + 스킬트리
- 헤더: 테마 전환 + 설정 버튼

### 2. 설정 모달 (탭 방식)
- 탭: [사운드] [통계] [업적]
- 사운드 탭: SFX/BGM ON/OFF + 볼륨 슬라이더
- 통계 탭: 기존 통계 내용
- 업적 탭: 기존 업적 내용

### 3. 상점 모달
- 기존 업그레이드 리스트
- 탭: [클릭] [자동] [특수] (업그레이드 카테고리)
- 버튼 그룹: 구매/MAX/초월 (현재 방식 유지)

### 4. 스킬트리 모달
- 헤더 아래에 [환생하기] 버튼 배치
- 용어 변경: 프레스티지 → 환생

### 5. 용어 변경
- 프레스티지 → 환생 (모든 UI 텍스트)

## 성공 기준
